AI-Powered Communication Assistant for the Elderly

@Bachelor project

AI NLP Promt engineering React Native AWS
View on GitLab

This project is an AI-powered communication assistant designed to simplify digital communication for elderly individuals who struggle with reading and writing. The app leverages Natural Language Processing (NLP), prompt engineering, and AI technologies, including the ChatGPT API and Text-to-Speech (TTS), to enhance user interaction.

The front-end is built using React Native, enabling cross-platform functionality on both iOS and Android. The back-end utilizes AWS Amplify, incorporating services like AWS Cognito for secure user authentication and AWS AppSync for real-time data management via GraphQL APIs.

NLP and AI play a critical role in the app, offering real-time text correction and sentence rephrasing through ChatGPT, while TTS technology converts text into speech, making communication more accessible. The app is designed with a human-centered approach, ensuring an intuitive interface tailored to the needs of elderly users.

The application has undergone extensive testing, with plans for future enhancements, including multi-language support. This project showcases expertise in AI, NLP, and modern development frameworks to create an inclusive and user-friendly communication tool.
