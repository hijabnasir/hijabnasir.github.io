Vending Machine Simulation Project

@OOP

Object-Oriented Programming System Design
View on GitHub

a software simulation of a vending machine, focusing on the management of various product types, including perishable and non-perishable items. The vending machine can handle a diverse range of products such as drinks, chocolates, pizzas, and chips, each assigned to specific slots. The system incorporates features to manage expiration and best-before dates, ensuring accurate inventory tracking.
