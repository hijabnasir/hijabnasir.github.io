Development of Numatix, a programming language designed for numerical computations, optimized for compilation into WebAssembly. The primary goal was to address the limitations of JavaScript in handling complex numerical tasks by leveraging WebAssembly's performance advantages. Numatix simplifies the process for developers to write high-level numerical code that can be compiled into efficient WebAssembly, enabling superior computational efficiency in web applications.

Designing and implementing key components of the Numatix language, including the syntax, semantics, and compiler phases. I focused on the creation of a straightforward yet powerful language that seamlessly integrates with JavaScript, allowing web developers to optimize performance-critical parts of their applications. The project involved rigorous testing and evaluation to ensure the language's reliability and effectiveness, culminating in the successful development of a functional compiler for Numatix.
