contributed to the development of Educado, an educational mobile application aimed at improving the lives of Brazilian waste pickers. The primary goal of the project was to create a user-friendly and accessible learning platform that could provide essential education and certifications to individuals with limited literacy skills. This project was part of a larger initiative involving multiple student groups, each focusing on different aspects of the application, including the mobile app, web app, and backend.

The Educado application was designed to work offline, considering the limited access to WiFi among the target users. Additionally, we ensured that the application was lightweight and could run on older-generation smartphones, making it accessible to the waste pickers. Through this project, I gained hands-on experience in mobile app development, UI/UX design, and Agile software development practices.
